# Atorch Console

> The console based Web Bluetooth.
>
> Note: Atorch Chinese is "炬为" (Pinyin: Jù wèi)

## Supported Meter

> Welcome to report more compatible hardware.

- USB Meter

  - Atorch UD18
  - Atorch T18 (UNTESTED)
  - Atorch DL24 (UNTESTED)

- AC Meter

  - Atorch AT24
  - Atorch AT3010 (UNTESTED)
